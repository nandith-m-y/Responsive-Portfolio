# 🌐 Responsive Portfolio Website

A modern, responsive, and professional personal portfolio website built using **HTML5, CSS3, JavaScript, and Bootstrap 5**. This project showcases my skills, projects, education, and contact information with a clean user interface and smooth user experience.

**Website:**

## 📂 GitHub Repository

**Repository:**
https://github.com/nandith-m-y/Responsive-Portfolio

# 📖 Overview

This project is a fully responsive personal portfolio website developed as part of a **Frontend Developer Internship Project**. It is designed to present my profile, technical skills, projects, educational background, and contact information in a professional and visually appealing manner.

The website is optimized for desktops, tablets, and mobile devices using responsive web design principles.

# 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/nandith-m-y/Responsive-Portfolio.git
```

Navigate to the project directory:

```bash
cd Responsive-Portfolio
```

Open `index.html` in your browser or use the **Live Server** extension in Visual Studio Code.
